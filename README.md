## Northern Luzon Weather App
A responsive web application that displays the current weather and 5-day forecast for cities in Northern Luzon, Philippines. The app fetches data from the OpenWeatherMap API and dynamically renders weather information, including temperature, humidity, and weather conditions with emojis.

## Features
- Search weather by city name
- Display current weather with temperature and humidity
- 5-day weather forecast
- Temperature in Celsius or Fahrenheit
- Emoji-based weather icons for easy readability
- Responsive design for desktop and mobile
- Clean, centered UI layout
